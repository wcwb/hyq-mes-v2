### 权限命名约定

```
{team_name}.{resource}.{action}

示例:
- mes_team_1.orders.view     # 查看订单
- mes_team_1.orders.create   # 创建订单
- mes_team_1.users.manage    # 管理用户
- mes_team_1.teams.admin     # 团队管理
```

#### ✅ 认证系统

- Laravel Sanctum令牌认证
- 安全的会话管理
- 设备管理功能

#### ✅ 权限管理
- 基于Spatie Permission的RBAC
- 团队级别的数据隔离
- 细粒度权限控制
- 超级管理员特权

#### ✅ 团队管理
- 多团队数据隔离
- 团队成员管理
- 团队权限分配
- 团队切换功能

#### ✅ 性能监控
